## 任务：每日动量报告

每个交易日生成《每日动量报告》，包含以下三个部分：

### 第一部分：Michael 的市场立场

根据以下指标给出操作建议：
- 恒指波幅指数 (VHSI)
- 恒生指数期货夜盘表现
- 港股通资金流向
- 整体市场情绪

**严格从以下三选一：**
| 选项 | 条件 |
|------|------|
| 激进买入 (Aggressive Buy) | 高信心，市场放量上涨趋势明显，北水持续流入 |
| 保守买入 (Conservative Buy) | 市场震荡或缺乏方向，仅参与特定板块或个股短线机会 |
| 持币观望 (Hold / Cash) | 市场过度波动或偏空（如 VHSI 飙升、期货大幅低水）|

### 第二部分：5% 观察名单

筛选 5 只具备上涨潜力的港股，每只需提供：
- **股票代码**（如 9988.HK）
- **胜率概率**（Win Probability，百分比）
- **选择理由**（技术形态、消息面催化、资金流向、板块联动）

### 第三部分：AI 大模型重点分析

当有智谱 (02513.HK) 或 MINIMAX-WP (00100.HK) 相关动态时，需分析：

**智谱 (02513.HK) 关注点：**
- 新一代旗舰模型调用量数据、技术评测排名
- 运营风险信号（服务限流、致歉信）
- 营收增速评估（2025上半年 1.91 亿元，+325%）
- 筹码结构（非禁售股仅占 8.5%，流通盘稀缺）

**MINIMAX-WP (00100.HK) 关注点：**
- 产品升级动态（MaxClaw、Expert 2.0）
- 海外收入占比（高达 70%）
- 成本优势（调用成本仅为竞品 1/10）
- 机构目标价（高盛 389 亿美元，瑞银 1000 港元）

## 使用方法

### 手动生成报告

在对话中直接要求生成报告：
```
"帮我生成今天的每日动量报告"
```

### 定时任务（可选）

配置 cron 任务每日开盘前发送：
- 港股开盘时间：9:30 AM (HKT)
- 建议报告时间：9:00 AM

## 数据来源

- 恒指波幅指数 (VHSI)：港交所
- 港股通资金流向：东方财富
- 个股数据：Wind / Bloomberg / 港交所披露
- 机构评级：各投行研报

---

## 注意事项

1. **高波动警示**：AI 大模型新股（智谱、MINIMAX）波动极大，单日可能暴涨 40% 或暴跌 20%
2. **流动性风险**：流通盘极小，少量资金即可撬动巨幅波动
3. **叙事驱动**：股价受消息面影响大，技术面参考需谨慎
4. **机构目标价**：仅作参考，实际走势可能与目标相差甚远
